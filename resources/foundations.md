# MBTI 通用认知框架

## 四字母含义
- **I/E**: 内向 (Introversion) / 外向 (Extraversion) — 注意力指向内部还是外部
- **N/S**: 直觉 (Intuition) / 实感 (Sensing) — 获取信息的方式
- **F/T**: 情感 (Feeling) / 思考 (Thinking) — 做决定的方式
- **P/J**: 感知 (Perceiving) / 判断 (Judging) — 对外界的态度

## 功能栈四格
- **主导功能**: 最擅长、最自然的认知方式
- **辅助功能**: 第二擅长的认知方式，平衡主导功能
- **第三功能**: 发展中的认知方式
- **劣势功能**: 最不擅长的认知方式

## 八功能简介
- **Ne**: 发散可能性，探索外部世界的模式
- **Ni**: 收敛洞察，内在的深层理解
- **Se**: 关注当下，感官体验
- **Si**: 依赖经验，细节记忆
- **Te**: 效率优先，外部逻辑组织
- **Ti**: 内部逻辑，精确分析
- **Fe**: 外部情感和谐，关注他人感受
- **Fi**: 内部价值判断，个人信念

## 轴张力
- 感知轴：Ne-Si / Ni-Se
- 判断轴：Te-Fi / Ti-Fe

## 心情联动原则
- 低心情 → 压力路径（劣势功能或回路循环）
- 高心情 → 辅助/第三功能舒适面
